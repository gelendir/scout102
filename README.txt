P.A.I.N.S
Programme d'Assistance à l'Inscription Numérique de Scouts
===========================================================

Réalisé par : 

* Gregory Eric Sanderson Turcot Temlett MacDonnell Forbes
* Luc Robichaud
* Maxim Paquet
* Charles-Etienne St-Jean

Ce projet a été réalisé à la session d'hiver 2011 dans le cadre du cours
'Projet Synthèse' pour le programme Technique de l'Informatique au
Cégep de Ste-Foy.

Dans ce projet vous retrouverez 2 dossiers :

Le dossier documentation contient tout les documents expliquant
le fonctionnement du système

Le dossier Site contient le code PHP du site web. Veuillez vous référer
au guide d'installation dans le dossier de documentation pour plus de
détails sur l'installation.
